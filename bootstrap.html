HTML

HTML (HyperText Markup Language) is the standard language used to create and structure content on the web.

CSS

CSS (Cascading Style Sheets) is used to style and layout web pages, controlling colors, fonts, spacing, and layout.

JavaScript

JavaScript is a programming language that brings interactivity to websites, such as animations, form validation, and dynamic content.

Bootstrap

Bootstrap is powerful frontend framework that simplifie designing responsive and modern websites with component and grid system.

Python

Python is a versatile and beginner-friendly programming language used in web development, data analysis, AI, automation, and more.
